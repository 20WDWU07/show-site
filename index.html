Web & UX Module 1

Module 1 is all about UX (User Experience) Design and runs for 10 weeks. UX research is the key to good design. Students strategize, research, design, develop and test. Users are involved at every stage. We learnt to develop websites based on user experience, prototype & test before coding. CSS style guides are prepared to follow best practices of Yoobee and web industry. Developers choose CSS Methodologies likeBEM OOCSS and use SASS as a CSS pre-processor to organize styling faster with partials and mixins. GitHub is used for version control. Implications to the Treaty of Waitangi are considered.

In this course we learned more about HTML, CSS and JavaScript techniques. The intention is to implement new features to keep on challenging ourselves as we dive deep into the web development world. We as students want to invest time and effort into our projects in order to gain experience and build a great portfolio.
